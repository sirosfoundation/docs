# SIROS ID Platform

Via the SIROS ID platform you can manage one or more digital identity credential manager services and connect them to issuers, verifiers and trust services. The SIROS ID Issuer and Verifier are built on open standards and work with **any OID4VCI/OID4VP-compatible wallet**—users can present credentials from the SIROS ID Credential Manager or any other compliant wallet.

For a quick start, users can access the SIROS ID web application at <a href="https://id.siros.org">id.siros.org</a> or use one of the SIROS native apps.

Here you will find all the information you need in order to manage, observe and integrate with your digital credential infrastructure.

The SIROS ID platform consists of the following parts:

* [SIROS ID Issuer and Verifier](/sirosid/issuers/concepts) – works with any compatible wallet, built on https://github.com/SUNET/vc
* [SIROS ID Credential Manager](/sirosid/reference/cm) – a reference wallet built on the [wwWallet](/opensource#wwwallet-project) project
* [SIROS ID Trust Services](/sirosid/trust)
* [Standards & Specifications](/sirosid/reference/standards) – comprehensive list of implemented standards
